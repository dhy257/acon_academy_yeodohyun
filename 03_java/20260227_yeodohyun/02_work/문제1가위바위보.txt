# 가위바위보

1. 입력 데이터
    - 변수명: user_info
    - 자료형: 정수형 (int)
    - 값의 범위: 1 (가위), 2 (바위), 3 (보)
2. 내부 생성 데이터
    - 변수명: computer_info
    - 자료형: 정수형 (int)
    - 값의 범위: 1 ~ 3 사이의 난수 (Random Number)
3. 출력 데이터 (Output)
    - 변수명: result_message
    - 자료형: 문자열 (String)
    - 출력 내용: '무승부', '사용자승리', '사용자패배' 중 하나
4. 처리
    1. 반복 시작 (while)
        - 조건: user_info == computer_info (입력과 난수가 같으면 무승부 >> 반복)
    2. 사용자로부터 user_info 입력 받음(1~3)
    3. Random 함수를 통해 computer_info 생성 (1~3)
    4. 승패 판단 (if-else)
        - 승리 조건: (1,3), (2,1), (3,2) 조합인 경우
        - 패배 조건: 그 외의 경우 (무승부 제외)
    5. 결과 출력 및 재시작 물어보기
    	- 재시작 >> 1로 돌아가기
	- 끝 >> 종료 출력 및 break
