객체메서드
1. 객체를 생성하고 >> 힙 사용
2. 객체이름.메서드() 형태로 사용하는 메서드
**객체의 속성(인스턴스 변수)을 직접 사용하거나 변경 가능**
this 키워드를 사용 >> 현재 객체의 정보를 사용할 수 있다

각방(객체)이 있고  리모컨(객체 메서드)으로 그 방에서의 조명등(인스턴스 변수)를 조정한다고 생각
this는 지금 있는 방

static메서드
1. 객체를 생성하지 않고
2. 클래스이름.메서드() 형태로 사용하는 메서드
**객체 변수는 직접 사용할 수 없음, static 변수는 사용 가능**
*클래스에 소속된 메서드*로 모든 객체가 공유 >> 공통 기능을 구현할 때 사용

대충 두꺼비집이라고 생각
