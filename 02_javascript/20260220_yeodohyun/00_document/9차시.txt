#이벤트 객체 사용하는 방법
 이벤트처리 핸들러 함수는 이벤트 객체를 제공받는다


#방법1)  태그에 이벤트 리스너 추가하기

HTML 태그에서 이벤트 속성(onclick, onchange 등)을 직접 사용하여 이벤트 리스너를 추가 
이벤트 핸들러 함수에서 명시적으로 event 객체를 매개변수로 전달받아 사용해야 함 

 
<button onclick="handleClick(event)">클릭하세요</button>
<script>
  function handleClick(event) {
    console.log(event.target); // 클릭된 요소
  }
</script>

##방법2)  addEventListener로 이벤트 리스너 추가하기
JavaScript 코드에서 DOM 요소에 이벤트 리스너를 추가할 때 addEventListener를 사용 
이벤트 핸들러 함수는 첫 번째 매개변수로 event 객체를 자동으로 전달받는다
 
const button = document.querySelector("button");
button.addEventListener("click", function (event) {
  console.log(event.target); // 클릭된 요소
});
