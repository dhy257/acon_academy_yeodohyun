
1. 자료형: 변수에 저장된 데이터의 type(숫자, 문자열, 논리형, undefined)
    let num = 7;
    console.log(typeof(num)); 사용하면 콘솔창에서 확인가능

자바스크립트는 동적타입언어로 실행시 저장되는 값에 의해서 타입이 결정됨


2. 변수: (변하는 값): 값을 저장하는 메모리공간
    let a;
    a=10;

    let b,c;
    b=100;
    c=10000;

3. 상수 (의미화 상수): 필요에 따라 변수의 값을 저장후 이용만 할 경우
    const PI = 3.14;

4. 리터럴(값 자체): 상수라고 부름
    let result = true;
    let num=20;
