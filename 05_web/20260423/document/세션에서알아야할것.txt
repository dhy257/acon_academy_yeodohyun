
세션얻기
HttpSession  session= request.getSession();            =>사용자의 세션저장소 얻기, 없으면 새로운 세션저장소 생성
HttpSession  session= request.getSession(false);      =>사용자의 세션저장소 얻기, 없으면   null반환


세션에 값 저장하기
session.setAttribute("key",  "value");

세션에 값 얻어오기
session.getAttribute("key");

세션에 값 삭제하기
session.removieAttribute("key");

세션삭제하기
HttpSession  session= request.getSession(false); 
session.invalidate(); 

하나의 사용자  = 브라우저

사용자정보는 서버에 접속하면 세션id가 자동으로 발급되고 
접속시 세션id가 자동으로 전달되면서 정보를 확인할 수 있다.
사용자 접속
→ 서버가 세션 생성
→ 세션 ID 발급 (JSESSIONID)
→ 브라우저 쿠키 저장
→ 다음 요청마다 자동 전송
→ 서버가 같은 사용자 식별
